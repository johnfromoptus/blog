My homelab runs on Ubuntu with a Docker/Traefik/Authelia stack for reverse proxying and authentication.

## Stack

- **OS**: Ubuntu Server
- **Reverse proxy**: Traefik
- **Auth**: Authelia
- **Containers**: Various self-hosted services

## Current focus

Setting up Claude Code with AWS Bedrock for unattended persistent sessions, using IAM long-lived keys for auth on the homelab side.
